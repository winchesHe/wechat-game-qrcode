# APK 反编译文件 Git 忽略说明

## 忽略规则

### 完全忽略
- `apktool_output/` - 整个目录（包含 11824 个 .smali 文件，可读性差）

### 忽略第三方库和框架代码
以下目录的 Java 源码会被忽略：
- `jadx_output/sources/android/` - Android 系统类
- `jadx_output/sources/androidx/` - AndroidX 库
- `jadx_output/sources/io/` - IO 相关库
- `jadx_output/sources/javax/` - JavaX 库
- `jadx_output/sources/kotlin/` - Kotlin 标准库
- `jadx_output/sources/kotlinx/` - Kotlin 扩展库
- `jadx_output/sources/mirrorb/` - 第三方库
- `jadx_output/sources/okhttp3/` - OkHttp 网络库
- `jadx_output/sources/okio/` - OkIO 库
- `jadx_output/sources/org/` - 其他第三方库
- `jadx_output/sources/p027/` - 混淆后的第三方库
- `jadx_output/sources/retrofit2/` - Retrofit 网络库
- `jadx_output/sources/com/google/` - Google 库
- `jadx_output/sources/com/muhua/cloud/` - 其他 muhua 模块
- `jadx_output/sources/com/muhua/wz/net/` - 网络工具类
- `jadx_output/sources/com/muhua/wz/util/` - 工具类
- `jadx_output/sources/com/muhua/wz/menu/` - 菜单相关
- `jadx_output/sources/com/muhua/wz/databinding/` - 数据绑定

### 忽略资源文件
- 所有图片：`.png`, `.jpg`, `.jpeg`, `.gif`, `.webp`, `.ico`
- 二进制文件：`.so`, `.dex`, `.apk`, `.jar`, `.class`
- 布局和资源：`drawable*/`, `mipmap*/`, `layout*/`, `color*/`, `anim*/`
- 库文件：`resources/lib/`

## 保留的文件

### 核心业务代码
- `jadx_output/sources/com/muhua/wz/*.java` - 主要业务逻辑
  - `MainActivity.java` - 主 Activity
  - `GameFragment.java` - 游戏 Fragment
  - `Constant.java` - 常量定义
  - `Utils.java` - 工具类
  - `bean/GameInfo.java` - 数据模型
  - 其他核心业务类

### 配置文件
- `jadx_output/resources/assets/package.json` - 游戏配置列表
- `jadx_output/resources/AndroidManifest.xml` - Android 清单文件
- `jadx_output/resources/res/values/*.xml` - 字符串、样式等资源

## 效果

忽略后，Git 只会跟踪：
- 约 20-30 个核心业务 Java 文件
- 几个关键的配置和资源文件

而不是原来的 19000+ 个文件。
